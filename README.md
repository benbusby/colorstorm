### From Source

- Install [Go](https://go.dev/doc/install)
- Run: `go build`
- Move `./colorstorm` into your `$PATH`

## Usage

```bash
$ colorstorm [optional jpg/png reference image]
```

#### Supported Editors
- Vim
- VSCode
- Sublime
